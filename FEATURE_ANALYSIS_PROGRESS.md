# Legado vs Flutter (iOS) Feature Analysis Progress

| 功能模組 | Legado 實作狀態 | Flutter (iOS) 完成度 | 不足之處 & 改進建議 | 狀態 |
| :--- | :--- | :--- | :--- | :--- |
| 書架 (Bookshelf) | 支援分組、多種排序、佈局切換、批量管理、URL/JSON 導入導出。 | 75% | 缺失 URL/JSON 書架導入導出功能；分組管理較簡易；缺乏 UI 細項開關（如未讀數顯示控制）。 | ✅ 已分析 |
| 閱讀器 (Reader) | 支援多種翻頁動畫、複雜排版、TTS、自動翻頁、替換規則、WebDAV 同步、換源。 | 80% | 缺失自動換源邏輯；缺乏 WebDAV 網路進度同步；內容處理細節（如反轉、刪除重複標題）尚待補足。 | ✅ 已分析 |
| 書源管理 (Source Manager) | 支援多種導入方式、編輯器、WebDAV 同步、分組管理、校驗功能、遷移、偵錯。 | 85% | 缺失 WebDAV 自動同步備份；編輯器功能深度（如偵錯控制台）尚待確認；缺乏網路書源庫搜尋。 | ✅ 已分析 |
| 詳情頁 (Book Detail) | 支援詳情載入、目錄抓取、資訊編輯、加入書架、換源、多選下載。 | 70% | 缺失書籍資訊手動編輯功能；目錄管理（如搜尋章節、排序）較弱；缺乏緩存管理細項。 | ✅ 已分析 |
| 搜尋 (Search) | 支援多源併發搜尋、範圍自定義、歷史記錄、熱搜詞、併行優化。 | 80% | 缺失搜尋範圍 (SearchScope) 指定功能；尚未實作整合書源熱搜詞；缺乏搜尋結果二次過濾（如依字數、狀態）。 | ✅ 已分析 |
| 本地導入 (Local Book) | 支援檔案瀏覽、多種格式 (EPUB, TXT, PDF 等)、智慧掃描、分組導入。 | 60% | 僅支援 TXT/EPUB；缺失智慧掃描頁面；目前解析後將內容全量存入資料庫，對大檔案效能較差。 | ✅ 已分析 |
| 替換規則 (Replace Rule) | 支援正則、分組、批量操作、排除範圍、導入導出。 | 50% | 目前僅支援基礎 CRUD 與排序；缺失分組管理；缺乏導入導出與即時測試功能；未實作規則套用範圍限制。 | ✅ 已分析 |
| RSS | 支援 RSS 來源管理、訂閱列表、文章閱讀、收藏功能。 | 65% | 基礎訂閱與閱讀已具備；缺失文章收藏夾功能；對圖片與 WebView 內容的優化處理尚待加強。 | ✅ 已分析 |
| 設定 (Settings) | 支援 WebDAV 備份、主題自定義、封面規則、效能細項、歡迎頁自定義。 | 70% | 具備核心 WebDAV 備份、主題與 TTS 管理；缺失自動備份邏輯；封面規則與效能調優細項尚待補足。 | ✅ 已分析 |
